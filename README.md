# web-project
assignment: a simple online shop using fastAPI, React...

## How to run
1. **clone the project**
2. **Setup environment variables:** Copy the example file to '.env':
	'cp .evn.example .env'
3. **Run with Docker:**
	'docker compose up --build'

## Management
- To shutdown project: 'docker compose down'
- to view logs: 'docker compose logs -f'
